# Real Estate Listing Copy Skill

## When to Use

Any request involving property marketing copy:

- "Write a listing description for [address]"
- "MLS description for this property"
- "Instagram post for new listing"
- "Open house flyer copy"
- "Just listed email"
- "Coming soon post"

## Input Requirements

Before writing, gather:

- **Property details**: Address, beds/baths/sqft, lot size, year built
- **Key features**: Upgrades, views, layout highlights, smart home, pool, etc.
- **Neighborhood**: School district, walkability, nearby amenities
- **Target buyer**: First-time, luxury, investor, family, downsizer
- **Price point**: Listed price or price range
- **Platform**: MLS, Zillow, Instagram, Facebook, email, flyer
- **Tone**: Professional, luxury, warm/inviting, investment-focused

If property details aren't provided, use agent-browser to pull them from Zillow/Redfin.

## Copy Frameworks

### MLS Description (250-500 words)

Structure:

1. **Hook** (1 sentence) — The single most compelling thing about this property
2. **Lifestyle paint** (2-3 sentences) — What living here feels like
3. **Key features** (bullet-style in prose) — Walk through the home logically: exterior → entry → living → kitchen → primary → secondary → outdoor → garage
4. **Neighborhood/location** (1-2 sentences) — Schools, commute, amenities
5. **Call to action** — Schedule showing, won't last

Rules:

- Never use: "boasts", "nestled", "turnkey", "must see", "won't last long" (overused)
- Use specific details over adjectives: "quartzite counters" not "beautiful counters"
- Mention room dimensions only if genuinely impressive
- Include smart home features, energy efficiency, recent upgrades with year
- Avoid making claims about value appreciation
- Comply with Fair Housing — no language about neighborhood demographics

### Social Media (Instagram/Facebook)

Structure:

- **Line 1**: Hook or emoji + status (🏡 Just Listed | Coming Soon | Open House)
- **Line 2-3**: 2-3 key highlights in short punchy lines
- **Line 4**: Price + location
- **Line 5**: CTA (Link in bio, DM for details, Open house date)
- **Hashtags**: 15-20 relevant hashtags (local + real estate)

Platform differences:

- **Instagram**: Visual-first, use line breaks, emoji accents (not excessive), carousel-friendly copy
- **Facebook**: Can be slightly longer, include link directly, community-focused angle
- **Twitter/X**: Ultra-concise, 1 feature + price + link

### Email (Just Listed / Open House)

Structure:

- **Subject line**: Address + 1 hook feature (e.g., "4BR Pool Home in Scottsdale — Just Listed at $625K")
- **Preview text**: Expand on the hook
- **Body**: 3-4 short paragraphs — hook, walkthrough highlights, neighborhood, CTA
- **CTA button**: "Schedule a Private Showing" or "See Full Details"

### Investment Property Copy

Different angle — lead with numbers:

- Cap rate, cash-on-cash return, gross rent multiplier
- Current rental income (if occupied)
- Market rent estimates
- Rehab estimate + ARV (if value-add)
- Occupancy rate for area
- 1% rule check

## Output Format

```
[Ready-to-use copy]

---
Format: [platform] | Target buyer: [type] | Character count: [N]
```

For MLS: include character count (many MLS systems cap at 1000-1500 chars).

## Quality Checklist

- [ ] No Fair Housing violations (no demographic descriptions of neighborhood)
- [ ] No guarantee language ("will appreciate", "best investment")
- [ ] Specific features over generic adjectives
- [ ] Accurate square footage and room counts
- [ ] Proper MLS abbreviations if MLS format (BR, BA, SF, etc.)
- [ ] CTA included
- [ ] Tone matches target buyer profile

## Rules

- If property details are missing, look them up before writing.
- Always include a character count for MLS descriptions.
- Never fabricate property features — only describe what's confirmed.
- Match the price point to the copy tone (don't write luxury copy for a starter home).
