# Real Estate Social Content Skill

## When to Use

Any social media content request:

- "Create a week of posts"
- "Just listed post for [address]"
- "Just sold announcement"
- "Market update post for [area]"
- "Neighborhood spotlight for [area]"
- "Content calendar for this month"
- "Video tour script for [property]"
- "Instagram reel ideas"

## Content Pillars

Every real estate agent's content should rotate through these:

| Pillar             | % of Content | Examples                                                       |
| ------------------ | ------------ | -------------------------------------------------------------- |
| **Listings**       | 25%          | Just listed, just sold, open house, price reduced              |
| **Market Intel**   | 25%          | Market updates, rate changes, inventory stats                  |
| **Community**      | 20%          | Neighborhood spotlights, local business features, events       |
| **Education**      | 15%          | Buyer tips, seller prep, investment basics, process explainers |
| **Personal Brand** | 15%          | Behind the scenes, client wins, testimonials, agent story      |

### Video Tour Script

```
[HOOK — 3 seconds]
"Wait until you see the kitchen in this [neighborhood] home..."

[INTRO — 10 seconds]
"Welcome to [address]. [Beds] beds, [baths] baths, [sqft] square feet, listed at $[price]. Let me show you around."

[ROOM-BY-ROOM — 15-30 sec each]
"Starting in the [living room/entry] — [describe the feeling, light, space]"

[CLOSE — 10 seconds]
"[Address], listed at $[price]. Link in bio for details or DM me to schedule a private showing."
```

## Platform-Specific Formatting

| Platform              | Format                | Length                | Best Practices                                                |
| --------------------- | --------------------- | --------------------- | ------------------------------------------------------------- |
| **Instagram Feed**    | Square or 4:5 image   | 150-300 words caption | Strong first line (before "more"), 20-30 hashtags in comments |
| **Instagram Reels**   | 9:16 vertical video   | 15-60 seconds         | Hook in first 3 seconds, text overlay, trending audio         |
| **Instagram Stories** | 9:16                  | Brief + interactive   | Polls, questions, quizzes, links                              |
| **Facebook**          | Landscape image/video | 100-250 words         | More personal tone, tag location, fewer hashtags (3-5)        |
| **LinkedIn**          | Portrait or landscape | 150-300 words         | Professional angle, market data, industry insights            |
| **TikTok**            | 9:16 vertical video   | 15-60 seconds         | Casual, trending sounds, fast-paced, personality              |
| **X/Twitter**         | Landscape image       | <280 chars            | Punchy, data-driven, thread for market updates                |
| **YouTube Shorts**    | 9:16 vertical         | <60 seconds           | Tour highlights, tips, market stats                           |

## Batch Generation

When asked for a batch of content:

1. **Confirm scope**: How many posts? What platforms? Any specific properties or topics?
2. **Check brand**: Load brand profile from `~/.openclaw/workspace/brands/` if available
3. **Generate varied content** across all pillars
4. **Vary hooks** — don't start every post the same way
5. **Include media direction** — describe what image/video should accompany each post

## Rules

- **Fair Housing**: No demographic descriptions of neighborhoods ("family-friendly", "young professionals area"). Describe amenities and features, not people.
- **No pressure language**: Avoid "won't last!", "act fast!", artificial urgency.
- **Hashtag strategy**: Mix broad (#RealEstate) with local (#ScottsdaleHomes) and branded (#[AgentTag]).
- **Engagement hooks**: Ask questions, use polls, encourage saves/shares.
- **Consistency**: Better to post 3x/week consistently than 7x/week for 2 weeks then nothing.
- **Photo direction**: When describing visual content, specify: subject, angle, lighting, mood.
- All content should be ready to post — no [brackets] except for personalization fields like [Agent Name].
